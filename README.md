# toby-go

> Use Toby with any Go application.

## Installation

```golang
import (
	tobyBot "github.com/toby-cloud/toby-go/bot"
	tobyMessage "github.com/toby-cloud/toby-go/message"
)
```

Then do `go get` to install the Toby package.
See usage in the `examples` folder.
